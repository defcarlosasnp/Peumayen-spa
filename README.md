# Casona en Arriendo

Este es un esqueleto básico en PHP para promocionar una casona para arriendo que incluye piscina, quincho y sauna.

## Estructura del Proyecto

- `index.php`: Página principal
- `header.php`: Encabezado con navegación
- `footer.php`: Pie de página
- `style.css`: Estilos CSS básicos
- `procesar_contacto.php`: Procesador del formulario de contacto

## Cómo usar

1. Coloca los archivos en un servidor web con soporte PHP.
2. Abre `index.php` en tu navegador.
3. Personaliza el contenido según tus necesidades.

## Funcionalidades

- Página de inicio con descripción de la casona
- Sección de características destacadas
- Formulario de contacto básico
- Diseño responsivo básico
